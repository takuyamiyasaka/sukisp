# SukiSp

このツールは話題をジャンルごとに投稿できるサービスです。

# Description
未ログイン
news 記事閲覧
話題閲覧機能
話題詳細閲覧
ページネーション

ログイン
話題新規投稿
話題編集
プロフィール編集
コメント機能
いいね機能
フォロー機能

使用している技術

インフラ
AWS(EC2,RDS,EIP)

ログイン機能
devise

写真アップロード機能
refile-mini_magick


その他
News API ActionMailer
